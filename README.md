ENG: This is a simple calculator with two text windows for entering numbers with four signed buttons and a text field with the output of the result.
To work with the site, you need to enter 2 arbitrary numbers, select a button with a mathematical action.
There are only 4 actions, these are sum(Сумма) - green button, difference(Разность) - red button, multiplication(Умножение) - blue button and division(Деление) - yellow button

RU: Это простой калькулятор с двумя текстовыми окошками для ввода чисел четырьмя подписанными кнопками и текстового поля с выводом результата.
Для работы с сайтом нужно ввести 2 произвольных числа, выбрать кнопку с математическим действием.
Действий всего 4, это сумма - зелёная кнопка, разность - красная кнопка, умножение - синяя кнопка и деление - жёлтая кнопка
